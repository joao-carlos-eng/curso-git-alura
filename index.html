Criação de um Repositório Git

Passo 1: Iniciar um repositório local


    $ mkdir meu-repositorio
    $ cd meu-repositorio
    $ git init

Passo 2: Adicionar e confirmar arquivos


    $ git add .
    $ git commit -m "Meu primeiro commit"

Passo 3: Criar um repositório remoto

Crie um repositório vazio em uma plataforma Git, como o GitHub, GitLab ou Bitbucket.

Passo 4: Associar o repositório remoto


    $ git remote add origin

Passo 5: Enviar as alterações para o repositório remoto


    $ git push -u origin master

Comandos Git adicionais

Aqui estão alguns outros comandos Git comumente usados:

  • git status - Verificar o status atual do repositório
  • git branch - Listar as branches (ramificações) disponíveis
  • git checkout -b nome-da-branch - Criar e mudar para uma nova branch
  • git pull origin master - Atualizar o repositório local com as alterações do repositório remoto
  • git log - Exibir o histórico de commits
